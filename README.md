# Eleventy

A base repository including [Eleventy](https://11ty.dev) with a basic layout.

Eleventy plugins included:

* [Navigation](https://www.11ty.dev/docs/plugins/navigation/)

## Run locally

1. `npm install`
2. `npm start`

## Build

1. `npm ci`
2. `npm run build`
